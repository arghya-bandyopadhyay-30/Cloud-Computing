Shared Responsibility Model

The House Analogy

  • When you build a house, it's the duty of the mason to install doors and locks properly. This is like AWS being responsible for the security OF the cloud (Compute, storage, DB, network software, etc.).
  • As the owner, it's your duty to use that lock for security. This is like the Customer being responsible for security IN the cloud (customer data, client-side data encryption).
